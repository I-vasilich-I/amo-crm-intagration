# AmoCRM widget service

## Installation

```bash
  npm install
```

## Change file name

```string
.env.example -> .env
```

## Running the application

  ```bash
  # run containers
  npm run docker:up

  # run prisma migration
  npm run prisma:migrate
  ```

## Direct database manipulations

  ```bash
  # open prisma studio
  npm run prisma:studio
  ```
